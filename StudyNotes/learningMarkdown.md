## 标题（两种方法）
# 一级标题：`# 文本`
## 二级标题：`## 文本`
### 三级标题：`### 文本`

主标题：
===
> 主标题
> `===`

副标题：
---
> 副标题
> `---`

## 正文：
正常格式不需要加任何其他符号
*斜体字*：`*文本*` 或者 `_文本_` (很明显要打中文的时候第一种更好)
**加粗字**：`**文本**` 或者 `__文本__`（吐槽同斜体）
***加粗斜体***：`***文本***` 或者 `___文本___`（吐槽同斜体）
~~删除线~~：`~~文本~~`
## 列表：
- 有序列表：`阿拉伯数字`+`.`+`空格`+`文字`
- 无序列表：`-`+`空格`+`文字`（使用`*`或者`+`也可以）

## 块引用：
`>`+`空格`+`文字`
可以写多行内容放入一个块中
> 来到这世上
> 好好儿活着
> 因为要死很久
## 代码：
+ 行内代码：使用反引号`` ` ``：`` ` ``+`文本`+`` ` ``（这里可能会有个问题就是不知道怎么在行内代码中显示`` ` ``,其实这里需要用两对`` ` ``来包裹一个`` ` ``：` `` ` `` ` `` ` `` `）
+ 多行代码块：使用三个反引号包裹代码` ``` `，并可指定语言（在一开始的` ``` `后写上你所指定的语言）
  比如运行代码要按`Enter`键：
  ```cpp
  #include<bits/stdc++.h>
  using namespace std;
  int main() {
    cout << "Hello Markdown!" << endl;
    return 0;
  }
  ```

## 分割线：
使用三个及以上的`*`、`-`或者`_`来创建分割线
如下：`***********`
******************************************
## 链接：
`[` + `链接文本` + `]` + `(` + `链接地址` + `)`
比如：[课程网站](https://docs.cpl.icu/)
![alt text](image-3.png)
## 表格：
使用`|`和`-`来创建表格。比如：
在第二行必须要有形如`|---|----|---|`的内容，从而划分列数。每一格中`-`的数量是不作要求的。
|学号|姓名|成绩|
|-|-|-|
|1|张三|100|
|2|李四|60|
