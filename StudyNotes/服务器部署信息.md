### Nginx 配置情况
**Nginx** 已验证安装，目录在：D:\LeStoreDownload\nginx

常用命令：
```
启动服务：	      winsw.exe start

停止服务：         winsw.exe stop

重启服务：         winsw.exe restart

查看状态：     	  winsw.exe status

卸载服务：	      winsw.exe uninstall

测试配置文件：	  nginx -t	            修改配置后先测试

重载配置：	      nginx -s reload	    不中断服务的情况下应用新配置
```

注意：配置了系统环境变量后，winsw.exe 相关的命令（winsw.exe start/stop/status）还是需要进入 Nginx 目录执行，因为 winsw 依赖同目录下的 winsw.xml 配置文件。

**一些背景信息**

IP 归属地与服务商：这个IP的注册服务商为 阿里云（Alibaba Cloud），地理位置通常被划定在亚太节点（如中国香港或新加坡）。

云服务器 IP 的特性：作为阿里云的公有云 IP，由于它随时可能被分配给不同的租户使用，或者作为 CDN、负载均衡或 WAF 的节点，因此经常存在“没有绑定域名”、“频繁更换绑定域名”或者是“泛解析状态”。

隐藏或内网映射：如果这是一个私人使用或用于纯接口（API）服务器的云主机 IP，所有者可能从未进行过公开的 DNS 域名解析，而是直接使用 IP 访问。
